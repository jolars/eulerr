# Compute geometries and label locations

Compute geometries and label locations

## Usage

``` r
setup_geometry(x, fills, edges, labels, quantities, n, id, merged_sets)
```

## Arguments

- x:

  an object of class 'euler'

- fills:

  fills

- edges:

  edges

- labels:

  labels

- quantities:

  quantities

- n:

  number of sets

- id:

  identity matrix

- merged_sets:

  which sets have been merged?

## Value

a list object with slots for the various objects
