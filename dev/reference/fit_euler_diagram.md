# Fit an Euler diagram using the eunoia Rust library.

Fit an Euler diagram using the eunoia Rust library.

## Usage

``` r
fit_euler_diagram(
  combo_names,
  combo_values,
  input,
  shape,
  loss,
  loss_eps,
  optimizer,
  n_restarts,
  extraopt_threshold,
  tolerance,
  max_sets,
  complement,
  seed,
  n_threads
)
```
