# Update list with input

A wrapper for
[`utils::modifyList()`](https://rdrr.io/r/utils/modifyList.html) that
attempts to coerce non-lists to lists before updating.

## Usage

``` r
update_list(old, new)
```

## Arguments

- old:

  a list to be updated

- new:

  stuff to update `x` with

## Value

Returns an updated list.
