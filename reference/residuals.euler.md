# Residuals of euler object

Residuals of euler object

## Usage

``` r
# S3 method for class 'euler'
residuals(object, dense = FALSE, ...)
```

## Arguments

- object:

  object of class `'euler'`

- dense:

  same meaning as in
  [`fitted.euler()`](https://jolars.github.io/eulerr/reference/fitted.euler.md)

- ...:

  ignored

## Value

A named numeric vector of residuals (input minus fit) keyed by
combination label.
