# Get the bounding box of an ellipse

Get the bounding box of an ellipse

## Usage

``` r
get_bounding_box(h, k, a, b = NULL, phi = NULL)
```

## Arguments

- h:

  x-coordinate for the center

- k:

  y-coordinate for the center

- a:

  radius or semi-major axis

- b:

  semi-minor axis

- phi:

  rotation

## Value

The bounding box as a list with xlim and ylim
