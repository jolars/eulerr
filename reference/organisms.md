# Organisms

Example data from the **VennMaster** package.

## Usage

``` r
organisms
```

## Format

A [matrix](https://rdrr.io/r/base/matrix.html) with 7 observations,
consisting of various organisms, and 5 variables: *animal, mammal,
plant, sea*, and, *spiny*, indicating whether the organism belongs to
the category or not.

## Source

<https://github.com/sysbio-bioinf/VennMaster/blob/master/data_examples/deploy/example1.list>

## Details

Note that this data is difficult to fit using an Euler diagram, even if
we use ellipses, which is clear if one chooses to study the various
overlaps in the resulting diagrams.
