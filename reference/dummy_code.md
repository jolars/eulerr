# Dummy code a data.frame

Dummy code a data.frame

## Usage

``` r
dummy_code(x, sep = "_", factor_names = TRUE)
```

## Arguments

- x:

  a data.frame

- sep:

  character for separating dummy code factors and their levels when
  constructing names

- factor_names:

  whether to include factor names when creating new names for dummy
  codes

## Value

A dummy-coded version of x.
